## Milestone Summary

**Key Decisions:**

- MIDI channels: mood=2, wombtone=3, brothers=4 (CBA defaults; avoids conflicts)
- HX Stomp .hlx file is user-created in HX Edit; rig.yaml entry only is in scope
- CBA dipswitches use 0 (off) / 127 (on) — never true/false/1
- Brothers gain_N_type: 0=boost, 1=od, 2=dist (0-indexed)
- Wombtone rate: 0 freezes phase (static cocked-wah); feed 96+ adds resonance
- Mood wet_channel: 1 = delay path (chorus-like)

**Issues Resolved:**

- Confirmed correct indentation for MC6 Switch B via grep before edit (initial attempt used wrong indentation from memory; corrected after verification)

**Issues Deferred:**

- WOMBTONE-02: True envelope-filter auto-wah (requires expression pedal + MC6 CC mapping; Wombtone has no onboard envelope follower)
- Phase 6 blocker: user must create `hlx/Mesa Rock.hlx` manually in HX Edit before rock scene can be fully tested end-to-end

**Technical Debt Incurred:**

- None (clean execution; all three phases single-plan, single-commit structure)
